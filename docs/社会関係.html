ヤクシマザルという生き物を知りたい

5 社会関係

社会関係と一口に言っても、扱う範囲は非常に広い。
ここでは、ニホンザル種内での群れ内の社会関係、群れ間の社会関係、他の動物との種間関係について紹介する。

5.1 群内の社会関係

5.1.1 社会構造

ニホンザルのメスは基本的に生まれた群れで生涯を過ごすが、オスは性成熟前後に生まれた群れから移出する（c.f. ニホンザルのオスの移出）。そのため、社会構造は母親・姉妹・叔母・姪などの母系の血縁個体集団が核となり、血縁個体と近接や毛づくろいをして一緒に過ごす時間が多い。

5.1.2 社会交渉

5.1.2.1 毛づくろい

5.1.2.2 音声

ヤクシマザルは連続して音声を発する際に、２回目のほうが長めで極めて高頻度で高くなり、２回目の音声の方がよく他個体の反応がある31。

生息環境の視界が限られた屋久島と視界が開けた金華山で比較した研究では、クーコールの頻度が屋久島のほうが高く、屋久島だけコドモよりオトナのほうが頻度が高くなる32。

5.1.3 社会性

ニホンザルはマカク属の中で最も専制的で不寛容である典型的な一種とされる33,34,35。それはたいていの敵対的交渉は一方方向で、仲直り行動はあまり起こらないためである36,37,38,39。しかし、ニホンザルには種内変異（intra-species variation）が知られている40,41,42。
例えば、小豆島集団では気温の影響を考慮してもサル団子をつくるサルの頭数が高崎山集団より多く、これは他個体に対する寛容性の高さが関わっている可能性を指摘している43。さらに大きなサル団子をつくることができる淡路島集団では、寛容性に関わるとされる紐引き協力実験44を行い、淡路島集団では高い割合で成功したものの勝山集団では低かった45。

寛容性の指標として使われている行動は、

  1. 反撃行動の頻度が高い(＝けんかの頻度そのものは低くはないが、肉体的接触を伴う激しい攻撃は少ない)
  2. 仲直り行動の頻度が高い
  3. 毛づくろいが近縁個体(or/and)高順位個体に偏らない
  4. 子守り行動の頻度が高い
  5. コドモの取っ組み合い遊びが長く続く

などが挙げられる34,39,46,47。

5.1.4 寛容的社会

ヤクシマザルの社会は寛容的社会（tolerant society）といわれている理由は、次の４つの事象が挙げられている41。

  1. 血縁者による連合の割合が低い48
  2. 末子優位が成り立たない49
  3. 出生率や毛づくろいの方向性に順位差がない48
  4. 仲直り行動の割合が高い48

他にも、寛容的社会であるとされる淡路島集団の特徴として、休息時の凝集性の高さがある。この理由として、3者以上からなる毛づくろい交渉と子守り行動の頻度が高いことが指摘されている50。

5.2 群間の社会関係

5.2.1 エンカウンター

群れ同士が出会うことをエンカウンター（encounter）という。

屋久島の西部地域海岸部に生息するヤクシマザルは、群れ密度が高く群間関係は敵対的であることが知られている25,51,52,48。

このことから、他群とのエンカウンターは自分の群れにとって不利な可能性がある。
チンパンジーのようなロングコールや、ネコ科動物のような臭い付け行動など、他群・他個体との接触回避行動がないニホンザルはどうしているのか？
音声プレイバックによる結果、他群の音声に対して、

  • 採食をやめ、
  • コンタクトコールを減らし、
  • 見回し行動を増加

させることで、他群と戦うか逃げるかを判断しているようだ53。

5.2.2 群れの消失・融合

屋久島では群れが消失し、基本的には一生涯を出自群で過ごすメスでも、別の群れに移籍することが知られている54。これは群れが遊動域を隣接群から防衛する最小限のサイズがあり、それを下回るとより大きな群れにメスが移籍することが示唆されている。

しかし、実はメスが移籍する前に群れのアルファオスがいなくなったと記述されている。メスにとってオスは他群との競合の際の”用心棒”的な役割が指摘されており[@@@]、自分たちを守ってくれるオスがいなくなったことがメスが他群に移籍した決定打になったかもしれない。

メスがアルファオスを頼りにしている様子は宮城県にある金華山のニホンザルで報告されている55,56。

5.3 種間関係

ヤクシカがヤクシマザルを利用していることの方が多い。

5.3.1 シカがサルに近づくのはなんで？

ヤクシカはヤクシマザルの糞を食べる57。
まるでハンバーグを食べるかのようにモグモグと。
だが、しかし、サル糞を食べるかどうかは地域差があるが、なぜそうなのかはわかっていない。

ヤクシカはヤクシマザルの採食に関連する声を聴いて食物の探索効率を上げている可能性がある58。
しかし、日中にシカを個体追跡してみた研究では、採食時間割合は全体の約25%で、そのうちサル関連の採食割合は多くても10%ほどに過ぎなかった59,60,61。

採食割合から見るとシカはサルばかりに頼って食べて生きているわけではないようだ。サル糞の食べ物としての質がどれだけ高いか気になるところだ。

5.3.2 サルがシカに近づくのはなんで？

ヤクシマザルはヤクシカの背中に乗っていたり、毛づくろいすることがある。
種間交尾行動を目撃したとする事例報告62があるが、この報告を読んでみると実際にはオスザルがメスジカの背中に乗っただけで交尾はしていなかった。 そのため、この行動は種間交尾行動とは言えないだろう。
同種の交尾相手がいないことで発散相手として異種のシカにまたがったとする交尾剝奪（mate deprivation）仮説が挙げられており、この仮説は挙げられた仮説の中で筆者の観察印象に最も近い。ちなみにサルがシカに乗る行動は交尾期に限らず観察できることを注記しておく。

サルとシカの種間関係に関する一般向けの本63もあるので、興味のある人は読んでみるとよいだろう。
